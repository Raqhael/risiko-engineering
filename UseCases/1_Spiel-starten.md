## Spiel starten

| Geschäftsprozess, Funktion   | Start des Spiels und Übergang von der Vorbereitung in den aktiven Spielzustand                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ---------------------------- |------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Ziel, Ergebnisse             | Spiel initialisieren.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| Akteure                      | Spieler.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Vorbedingungen               | Alle Spieler sind bereit.                                                                                                                                                                                                                                                                                                                                                          |
| Auslösendes Ereignis         | Alle Spieler sind dem Spiel beigetreten. Das Spielsystem validiert den Spielstart                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| Nachbedingung bei Erfolg     | Das Spiel wurde erfolgreich initialisert und die erste Runde hat begonnen.                                                                                                                                                                          |
| Nachbedingung bei Fehlschlag | Rückkehr zum vorinitialisierten Zustand.                                                                                          |
| Eingehende Daten             | Anzahl der teilnehmenden Spieler.                                                                                                                            |
| Ausgehende Daten             | Startspieler, Aktueller Spielzustand, Ausgeteilte Missionskarten.                                                                                                                   |
| Ablauf                       | 1. Spieler bestimmen: Reihenfolge zufällig festlegen.<br>2. Startterritorien werden ausgeteilt . In jedem zugewiesenen Territorium wird eine Truppe stationiert.<br>3. Missionskarten zuweisen.  |
| Erweiterungen                | -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| Alternativen                 | 1. Falls zu wenige Spieler teilnehmen: Rückkehr der Spieler zur Lobby.<br>2. Falls ein Fehler während der Initialisierung auftritt: Neuversuch oder Abbruch mit Fehlermeldung.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
